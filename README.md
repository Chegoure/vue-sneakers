# Vue Sneakers — Интернет-магазин кроссовок

Учебный проект по разработке полноценного SPA-приложения на Vue 3. Цель — изучить работу с Composition API, передачу данных между компонентами, роутинг и взаимодействие с API.

## 🛠 Стек технологий

- **Vue 3 (Composition API)** — основной фреймворк.
- **Tailwind CSS** — верстка и стилизация.
- **Vite** — быстрая сборка проекта.
- **Prettier** — форматирование кода.
- **Axios** — HTTP-запросы к API.

## 🚀 Что уже реализовано:

- [x] Развернута база проекта на Vite.
- [x] Настройка Tailwind CSS.
- [x] Верстка компонент Header (шапка сайта).
- [x] Создание карточки товара и сетки (Card, CardList).
- [x] Подключение backend API через Mokky.dev.
- [x] Получение и отображение списка товаров с сервера.
- [x] Реализация системы фильтрации и поиска.
- [x] Реализация системы добавления товара в "избранное".
- [ ] Реализация логики корзины.
- [ ] Добавление роутинга.

## Текущий прогресс

- Получил список кроссовок с backend API.
- Настроил передачу данных в `CardList`.
- Добавил отображение карточек товаров через `v-for`
- Реализовал фильтрацию товаров по цене и названию.
- Добавил поиск товаров через input.
- Настроил повторный запрос к API при изменении фильтров через `watch`.
- Добавил query-параметры в Axios-запросы.
- Реализовал систему избранного.
- Настроил получение избранных товаров с сервера при загрузке приложения.
- Добавил добавление и удаление товаров из избранного через API.
- Реализовал смену иконки в зависимости от состояния товара.
